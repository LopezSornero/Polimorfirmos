Escritura de Métodos

Un método es un bloque de código que realiza una tarea específica. Los métodos se pueden usar para:

  • Evitar la duplicación de código.
  • Organizar el código en unidades más pequeñas y manejables.
  • Hacer que el código sea más legible y fácil de entender.

    • Partes de un método:

    • Modificador de acceso: Define quién puede acceder al método.
    • Tipo de retorno: Indica el tipo de dato que devuelve el método.
    • Nombre del método: Identifica al método.
    • Parámetros: Son valores que se pasan al método cuando se invoca.
    • Cuerpo del método: Contiene las instrucciones que se ejecutan cuando se invoca el método.

    Ejemplo de método:

            public void Saludar()
        {
            Console.WriteLine("Hola!");
        }

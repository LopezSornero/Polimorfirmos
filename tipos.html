Tipos de Polimorfismo

Existen dos tipos principales de polimorfismo:

  • Polimorfismo por inclusión:

    Se produce cuando una clase hereda de otra clase. La clase hija puede tener métodos con el mismo nombre que la clase padre, pero con diferentes implementaciones.
Ejemplos de Polimorfismo por inclusión:
  • Una clase `Animal` con un método `comer()`. Las clases hijas `Perro`, `Gato` y `Vaca` pueden tener diferentes implementaciones del método `comer()`.
  • Una clase `Figura` con un método `calcularArea()`. Las clases hijas `Cuadrado`, `Triángulo` y `Círculo` pueden tener diferentes implementaciones del método `calcularArea()`.

  • Polimorfismo por herencia:

    Se produce cuando se utiliza una variable de tipo padre para almacenar una referencia a un objeto de tipo hijo. La variable de tipo padre puede acceder a los métodos del objeto hijo que son públicos o heredados.
Ejemplos de Polimorfismo por herencia:
  • Una variable de tipo `Animal` que puede almacenar una referencia a un objeto de tipo `Perro`, `Gato` o `Vaca`.
  • Una variable de tipo `Figura` que puede almacenar una referencia a un objeto de tipo `Cuadrado`, `Triángulo` o `Círculo`.
